# Perulangan Bersarang (Nested Loop)
print("Pola segitiga bintang:")
baris = 5
for i in range(1, baris + 1):
    for j in range(i):
        print("*", end=" ")
    print()

print("Tabel perkalian 1-3:")
for i in range(1, 4):
    for j in range(1, 4):

    print()

Algoritma, Program, dan Struktur Dasar

Pembahasan mengenai konsep algoritma dan program, tiga struktur dasar algoritma, serta fungsi bahasa pemrograman.

1. Algoritma dan Program

Algoritma adalah urutan langkah-langkah yang logis dan sistematis untuk menyelesaikan suatu masalah. Program adalah implementasi algoritma ke dalam bahasa pemrograman sehingga dapat dijalankan oleh komputer.

Kesimpulan: Algoritma adalah rencana penyelesaian masalah, sedangkan program adalah bentuk nyata dari algoritma yang dapat dijalankan komputer.

Contoh Notasi Kalimat Deskriptif
contoh
x = 4
y = (3 x 4) + 6
y = 12 + 6
y = 18
2. Tiga Struktur Dasar Algoritma

a. Struktur Sekuensial (Sequence) — Struktur yang menjalankan instruksi secara berurutan dari awal sampai akhir tanpa ada percabangan.

Contoh: Input nilai, Hitung nilai, Tampilkan hasil.

b. Struktur Pemilihan (Selection) — Struktur yang digunakan untuk memilih tindakan berdasarkan suatu kondisi.

contoh
Jika nilai >= 75
    Tampilkan "Lulus"
Jika tidak
    Tampilkan "Tidak Lulus"

c. Struktur Perulangan (Looping) — Struktur yang digunakan untuk mengulang suatu proses beberapa kali.

contoh: menampilkan angka 1 sampai 5
1
2
3
4
5
3. Fungsi Bahasa Pemrograman

Bahasa pemrograman berfungsi untuk memberikan instruksi kepada komputer agar dapat menjalankan tugas tertentu sesuai keinginan pengguna.

Python — pengembangan aplikasi, data science, AI, dan otomatisasi.

Java — aplikasi Android, desktop, dan sistem enterprise.

C++ — pembuatan game, sistem operasi, dan aplikasi berperforma tinggi.

JavaScript — membuat website menjadi interaktif.

PHP — pengembangan website dan aplikasi berbasis web.

Kesimpulan: Bahasa pemrograman berfungsi sebagai media komunikasi antara manusia dan komputer untuk membuat berbagai aplikasi, website, maupun sistem informasi.

Tipe Data, Variabel, Array, dan Operator

Pembahasan mengenai tipe data primitif/non-primitif, analisis kesalahan penulisan variabel, deklarasi array, dan operator aritmatika dalam Java.

1. Tipe Data Primitif vs Non-Primitif

Tipe Data Primitif: tipe data bawaan dari bahasa pemrograman yang hanya menyimpan satu nilai tunggal langsung di dalam memori, ukurannya tetap, dan tidak memiliki method bawaan. Contoh: int, float, double, char, boolean.

Tipe Data Non-Primitif (Referensi): tipe data yang dibuat oleh programmer atau struktur data kompleks yang menyimpan referensi (alamat memori) ke objek sebenarnya, ukurannya dinamis, dan memiliki method bawaan. Contoh: String, Array, Class, Interface.

2. Analisis Kesalahan Penulisan Variabel
kode asli (salah)
Int 5angka;
String nama siswa;
Int static;

Int 5angka; — Kesalahan 1: penulisan tipe data harus huruf kecil (int), Java bersifat case-sensitive. Kesalahan 2: nama variabel tidak boleh diawali angka. Perbaikan: int angka5; atau int limaAngka;

String nama siswa; — Kesalahan: nama variabel menggunakan spasi, harus ditulis menyambung. Perbaikan: String namaSiswa; atau String nama_siswa;

Int static; — Kesalahan 1: penulisan int harus huruf kecil. Kesalahan 2: kata static adalah keyword resmi Java, dilarang dipakai sebagai nama variabel. Perbaikan: int variabelStatic; atau int status;

3. Deklarasi Tipe Data Array
contoh deklarasi array (Java)
// Contoh 1: Deklarasi array kosong dengan ukuran 5
int[] kumpulanAngka = new int[5];

// Contoh 2: Deklarasi array sekaligus memberi nilai
String[] daftarMahasiswa = {"Andi", "Budi", "Cici"};
4. Operator Aritmatika dalam Java

+ (Penjumlahan) — menambahkan dua nilai. Contoh: 5 + 3 menghasilkan 8.

- (Pengurangan) — mengurangi nilai pertama dengan nilai kedua. Contoh: 5 - 3 menghasilkan 2.

* (Perkalian) — mengalikan dua nilai. Contoh: 5 * 3 menghasilkan 15.

/ (Pembagian) — membagi nilai pertama dengan nilai kedua. Contoh: 10 / 2 menghasilkan 5.

% (Modulo) — mengambil sisa hasil pembagian.

5. Implementasi Operator Pembagian (Java)
Main.java
public class Main {
    public static void main(String[] args) {
        // Deklarasi variabel angka yang akan dibagi
        double totalNilai = 90.0;
        double jumlahSiswa = 4.0;

        // Menggunakan operator pembagian ( / )
        double hasilRataRata = totalNilai / jumlahSiswa;

        // Menampilkan hasil ke layar
        System.out.println("Total Nilai: " + totalNilai);
        System.out.println("Jumlah Siswa: " + jumlahSiswa);
        System.out.println("Hasil Pembagian (Rata-rata): " + hasilRataRata);
    }
}
Output
Total Nilai: 90.0
Jumlah Siswa: 4.0
Hasil Pembagian (Rata-rata): 22.5

Implementasi Notasi Algoritma — Tarif Parkir

Studi kasus penentuan tarif parkir kendaraan menggunakan notasi kalimat deskriptif, pseudocode, dan diagram alur (flowchart).

Sebuah tempat parkir menerapkan kebijakan tarif: kendaraan yang parkir lebih dari 2 jam dikenakan tarif Rp5.000 per jam, sedangkan kendaraan yang parkir sama dengan atau kurang dari 2 jam dikenakan tarif tetap Rp3.000. Buatlah algoritma untuk menghitung total tarif parkir yang harus dibayar.

1. Notasi Kalimat Deskriptif
Algoritma_HitungTarifParkir
1. Masukkan jenis kendaraan
2. Masukkan lama waktu parkir (dalam jam)
3. Cek apakah lama parkir lebih dari 2 jam
   - Jika YA: Hitung tarif = lama parkir x Rp5.000
   - Jika TIDAK: Tarif parkir tetap sebesar Rp3.000
4. Tampilkan jenis kendaraan dan total tarif parkir yang harus dibayar
2. Notasi Pseudocode
PROGRAM HitungTarifParkir
DEKLARASI
  jenis_kendaraan : string
  lama_parkir : real
  tarif : real

ALGORITMA
  Input data dari pengguna
  read(jenis_kendaraan)
  read(lama_parkir)

  Penentuan besaran tarif parkir
  JIKA lama_parkir > 2 MAKA
     tarif <- lama_parkir * 5000
  LAIN
     tarif <- 3000
  AKHIR JIKA

  Menampilkan hasil perhitungan
  write("Jenis Kendaraan: ", jenis_kendaraan)
  write("Total Tarif Parkir: Rp", tarif)
End

Array (Larik)

Pembahasan mengenai pengertian array, array satu dimensi, cara deklarasi, tipe data, dan macam-macam array.

1. Apa yang Dimaksud dengan Array?

Array (Larik) adalah sebuah struktur data yang digunakan untuk menyimpan sekumpulan data atau elemen dengan tipe data yang sama (homogen) di dalam satu variabel. Setiap elemen di dalam array dapat diakses secara langsung melalui sebuah indeks (posisi angka). Di sebagian besar bahasa pemrograman, indeks array selalu dimulai dari angka 0.

2. Array 1 Dimensi

Array 1 dimensi adalah jenis array yang paling sederhana, di mana elemen-elemennya disusun secara linear atau dalam satu baris tunggal. Bayangkan array 1 dimensi seperti sebuah loket panjang yang berjejer, di mana setiap kotak loket memiliki nomor kunci (indeks) tersendiri untuk menyimpan satu barang.

3. Cara Mendeklarasikan Array

Cara mendeklarasikan array bergantung pada bahasa pemrograman yang digunakan. Secara umum, deklarasi membutuhkan komponen: Tipe Data, Nama Array, dan Ukuran/Jumlah Elemen.

4. Tipe Data untuk Mendeklarasikan Array

Array dapat menggunakan tipe data apa saja, baik tipe data dasar (primitif) maupun tipe data bentukan, dengan syarat seluruh elemen di dalam satu array harus memiliki tipe data yang sama. Contoh tipe data yang sering digunakan: int/integer (angka bulat), float/double (angka pecahan), char (karakter tunggal), string (teks), boolean (nilai benar/salah).

5. Macam-macam Array

Array 1 Dimensi (One-Dimensional Array): array yang terdiri dari satu baris atau satu kolom data saja.

Array 2 Dimensi (Two-Dimensional Array): array yang terdiri dari baris dan kolom, sering digambarkan seperti tabel atau matriks, memiliki dua indeks (indeks baris dan indeks kolom).

Array Multi-Dimensi (Multi-Dimensional Array): array yang memiliki lebih dari dua dimensi (misalnya 3 dimensi atau lebih), contoh visualisasinya seperti kubus (memiliki panjang, lebar, dan ruang/kedalaman).
